# flora

Botanical country data. Each country grows into a flower — population shapes the stem, flag colors bloom into petals.

**Visual Style:** Light — Daylight (warm cream paper background, dark brown organic strokes, soft coral accent)

**Data Source:** REST Countries API

**How it works:**
- Fetches country data (population, area, flag colors)
- Each country becomes a unique flower structure
- Stem height proportional to population
- Petal count and colors from area and flag
- Tap to cycle through 15 countries
- Smooth animated growth transitions

**Live:** https://parthchandak02.github.io/flora/

**Tech:** Native Canvas 2D, single HTML file, zero dependencies.
